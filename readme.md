# 대전시민천문대 홈페이지 리뉴얼

#### 팀원 : 이은솔, 유상범, 이상희, 서태규

### 🖐 프로젝트 목적

1. 홈페이지를 통해 예약하는 소비자들에게 편리한 예약 서비스 제공
2. 홈페이지 가독성 개선

### 🖐 프로젝트 내용

대전시민천문대는 index와 예약페이지를 따로 운영하고 있어 소비자가 예약을 하기위해 홈페이지에서 예약창을 찾으러 다니거나 또는 예약창에서도 찾을 수 없는 탭이 있어 체험율이 낮아질 수 있는 부분들을 indexpage로 배치하여 소비자가 보다 예약에 접근하는것이 보다 편리하도록 개선
